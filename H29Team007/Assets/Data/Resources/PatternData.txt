// パターンデータ書式
// パターン名;動物名,...;ポイント取得割合（％）
// 動物は３つ以上でも大丈夫です。
// 例
// ３匹の子豚;buta,buta,buta;50
// 豚が４８０円だった場合
//　計算式
// 豚のポイント 480 / 100 = 4.8 -少数点切り上げ->　5ポイント
// (5 + 5 + 5) * 50 / 100 -小数点切り上げ->  = 8ポイント取得
// 
// 
// 
// 
// 的な感じです。
// 
// 要素
// arai
// buta
// gyo
// hera
// hitsu
// kaji
// niwa
// tougyu
// ushi
// same

３匹の子豚;buta,buta,buta;50
三種の肉詰め;buta,ushi,niwa;50
海の幸詰め;gyo,kaji,same;50
危険動物;hera,kaji,tougyu,same;50
